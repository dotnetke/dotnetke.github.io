# Getting Started with .NET

Welcome to the .NET Fundamentals course! This comprehensive series will take you through the essential concepts of .NET development.

## Course Overview

In this course, you'll learn:
- What is .NET and the .NET ecosystem
- Setting up your development environment
- C# programming fundamentals
- Object-oriented programming concepts
- Working with packages and dependencies
- Building and running .NET applications

## Prerequisites

- Basic understanding of programming concepts
- A computer running Windows, macOS, or Linux
- Enthusiasm to learn! 🚀

## What You'll Build

Throughout this course, you'll build several projects:
1. **Console Calculator** - Learn basic C# syntax and operations
2. **Task Manager** - Understand classes, objects, and file I/O
3. **Simple API Client** - Work with HTTP requests and JSON
4. **Unit Testing** - Learn to test your code effectively

## Course Structure

Each lesson in this series includes:
- 🎯 **Learning objectives**
- 📖 **Conceptual explanations**  
- 👨‍💻 **Hands-on coding examples**
- 🏋️‍♀️ **Practice exercises**
- 🎥 **Video demonstrations** (coming soon)

## Setting Up Your Environment

Before we begin, make sure you have:

### Required Software
1. **.NET SDK** (version 8.0 or later)
   - Download from [dot.net](https://dot.net)
2. **Code Editor** (choose one):
   - Visual Studio Community (Windows/Mac)
   - Visual Studio Code (Cross-platform)
   - JetBrains Rider

### Verify Installation

Once installed, open your terminal and run:
```bash
dotnet --version
```

You should see version 8.0.0 or higher.
