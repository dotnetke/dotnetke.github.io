# Building Web APIs with ASP.NET Core

Learn to create powerful, scalable web APIs using ASP.NET Core. This course focuses on practical skills needed to build production-ready APIs for modern applications.

## Course Overview

This hands-on course covers:
- Creating RESTful APIs with ASP.NET Core
- Data persistence with Entity Framework Core
- Authentication and authorization
- API documentation with Swagger/OpenAPI
- Testing your APIs
- Deployment to Azure and other platforms

## What You'll Build

Throughout this course, you'll build a **Kenyan Business Directory API** that includes:
- Business listings with locations
- User authentication and authorization
- Review and rating system  
- Search and filtering capabilities
- Admin dashboard functionality

This project reflects real-world scenarios relevant to the Kenyan tech ecosystem.

## Prerequisites

Before starting this course:
- Complete our [.NET Fundamentals](/courses/dotnet-fundamentals/getting-started) course
- Basic understanding of HTTP and REST principles
- Familiarity with C# programming
- Basic knowledge of databases (helpful but not required)

## Development Environment

You'll need:
- .NET 8 SDK or later
- Visual Studio, VS Code, or Rider
- SQL Server Express or LocalDB (we'll cover setup)
- Postman or similar API testing tool

## Course Modules

### Module 1: API Fundamentals
- Setting up ASP.NET Core Web API project
- Understanding controllers and actions
- HTTP methods and status codes
- Request/response handling

### Module 2: Data Management  
- Entity Framework Core setup
- Database design and migrations
- CRUD operations
- Seed data for testing

### Module 3: Security
- Authentication with JWT tokens
- Authorization policies
- User management
- Protecting sensitive endpoints

### Module 4: Advanced Features
- Input validation and error handling
- Logging and monitoring
- Caching strategies
- API versioning

### Module 5: Testing & Deployment
- Unit testing API endpoints
- Integration testing
- Deployment to Azure App Service
- CI/CD with GitHub Actions
