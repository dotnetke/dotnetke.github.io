# Welcome to .NET Kenya Courses

Learn .NET development with our comprehensive courses and learning series designed specifically for developers in Kenya and the broader East African region.

## Featured Learning Series

Our courses are designed to take you from beginner to advanced levels in various .NET technologies. Each series includes hands-on examples, real-world projects, and community support.

### 🚀 Getting Started with .NET
Perfect for beginners who want to start their journey with .NET development.

### 🌐 ASP.NET Core Web Development
Learn to build modern web applications using ASP.NET Core.

### 📱 Cross-Platform Development
Explore .NET MAUI for building cross-platform mobile and desktop applications.

### ☁️ Cloud Development with Azure
Deploy and scale your .NET applications in the cloud.

## Learning Path

1. **Foundation**: Start with C# basics and .NET fundamentals
2. **Web Development**: Build web applications with ASP.NET Core
3. **Database Integration**: Work with Entity Framework Core
4. **Testing**: Learn unit testing and integration testing
5. **Deployment**: Deploy applications to various platforms
6. **Advanced Topics**: Microservices, performance optimization, and more
